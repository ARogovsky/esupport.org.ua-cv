# Security measures implemented:
# - Content Security Policy (CSP) headers
# - Input sanitization on chat API
# - Prompt injection defense (multi-layer: keyword detection, canary tokens, fingerprinting, anti-extraction)
# - Rate limiting on API endpoints
# - No user data stored — stateless chat sessions
# - 71 automated evals including security/jailbreak tests
# - AWS Bedrock with SigV4 authentication
# - Edge runtime for API endpoints (Vercel)
